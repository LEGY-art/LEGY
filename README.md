# LEGY
Soy LEGY, músico, productor, arreglista y compositor cubano. Fundador de iLEGY Musicx, una compañía musical en crecimiento. Ganador de tres ediciones del Festival del Tambor y formado en el Conservatorio Guillermo Tomás. Comparto mi música, proyectos y pasión con el objetivo de conectar con el mundo a través del arte.
